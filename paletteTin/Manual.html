Palette Tin

Palette Tin is a Krita Docker plugin designed to help with color mixing and palette management. Inspired by the Art Academy mixing interface, it aims to make painting more fun by reducing the need for micromanaging color picking and mixing outside the canvas.

Features

Color Mixing Modes

  1. Spectral: Uses Kubelka-Munk theory (via [spectral.js](https://github.com/rvanwijnen/spectral.js))
  2. Weighted Average: Mixes two colors using a weighted average.
  3. Hybrid: Combines spectral (60%) and weighted average (40%) for slightly less saturated results.
  4. Overlay: Blends colors based on lightness (Screen if light, Multiply if dark), useful when highly saturated mixes are needed
  5. Sat Val: Transfers saturation and value from the foreground while preserving the hue.
  6. Sat Val okhsl: Transfers okhsl saturation and lightness, retaining perceptual values. (via [Bj√∂rn Ottosson OKHSL](https://bottosson.github.io/misc/ok_color.h))

Palette Management

  1. Saves the palette directly into the working document as a note.
  2. Stores palettes for later use in other documents using a custom format.

Extended Mixing Options

  1. Mix paint outside the canvas.
  2. Includes a scratchpad for testing colors off-canvas.

Launching the Docker

  1. Navigate to Settings -> Dockers and enable Palette Tin to display the Docker.

Usage

  1. Mixing: Select a mixing mode from the dropdown, adjust the mix rate, and click on the palette grid to mix colors.
  2. Scratchpad: Test and experiment with colors off the main canvas.
  3. Palette Storage: Save and load your palettes directly within the document for future use.
